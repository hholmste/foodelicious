foodelicious
============

Prerequisites:
- Java 1.7 or above
- Mongodb installed

Get Started
-----------

1. git clone https://github.com/mariusbreivik/foodelicious.git
2. mvn package
3. java -jar target/foodelicious.jar server ./foodelicious.yml
4. open http://localhost:8080/  with a prefered browser
